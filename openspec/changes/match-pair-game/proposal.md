# Proposal: Match-Pair Game (对对碰游戏)

## 背景
用户需要一个经典的对对碰（翻牌消除）游戏，用于在浏览器中运行。

## 目标
创建一个单文件的 HTML 对对碰游戏，包含翻牌逻辑、配对检测、计分和重置功能。

## 范围
### 包含
- 4x4 的游戏网格（16张牌，8对）
- 翻牌动画效果
- 配对成功/失败逻辑
- 计分系统（尝试次数）
- 游戏结束提醒
- 重新开始功能

### 验收标准
- ✅ 点击牌面可以翻转
- ✅ 翻开两张相同的牌则保持翻开状态
- ✅ 翻开两张不同的牌则自动翻回
- ✅ 所有牌配对成功后显示胜利消息
- ✅ 界面响应式，适配不同屏幕
