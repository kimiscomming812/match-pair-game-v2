### CSS特性
- **渐变**: linear-gradient（紫色渐变）、radial-gradient（动态背景）
- **玻璃态**: backdrop-filter: blur(20px) + 半透明背景
- **动画**: keyframes动画（脉动、浮动、光晕、粒子）
- **3D翻转**: transform: rotateY(180deg) + preserve-3d
- **响应式**: @media媒体查询 + flex布局

### JavaScript模块

#### 1. CONFIG（游戏配置）
```javascript
difficulty: {
  '3x3': { pairs: 6, size: 9 },
  '4x4': { pairs: 8, size: 16 },
  '5x5': { pairs: 12, size: 25 }
}
emojis: 12个游戏符号
baseScore: 10分
comboMultiplier: 1.5倍
hintPenalty: 5分
timeBonus: 0.5分/秒
```

#### 2. AudioSystem（音效管理）
- 使用Web Audio API（AudioContext）
- 频率控制 oscillators
- 音量控制 gainNodes
- 7种不同音效类型

#### 3. ParticleSystem（粒子效果）
- Canvas 2D渲染
- 爆炸效果：30个粒子，重力下落
- 烟花效果：三波，每波20-30个粒子
- 60fps动画循环

#### 4. Leaderboard（排行榜系统）
- localStorage存储
- JSON序列化
- 排序算法（分数优先，时间次之）
- Top 10限制

#### 5. Game Logic（游戏逻辑）
- 卡片状态管理
- 匹配检测
- 连击计算
- 成绩结算
- 游戏流程控制
