# Proposal: Snake Game (贪食蛇游戏)

## 背景
用户需要一个经典贪食蛇游戏，用于在浏览器中运行。

## 目标
创建一个单文件的 HTML 贪食蛇游戏，包含蛇身移动、食物生成、碰撞检测和计分功能。

## 范围
### 包含
- 20x20 的游戏网格
- 蛇身移动（上下左右控制）
- 食物随机生成
- 碰撞检测（撞墙/撞自己死亡）
- 计分系统（吃食物得分）
- 游戏结束和重新开始功能
- 暂停/继续功能

### 验收标准
- ✅ 使用方向键控制蛇移动
- ✅ 蛇吃到食物后变长，得分增加
- ✅ 撞墙或撞自己时游戏结束
- ✅ 显示当前分数和最高分
- ✅ 支持暂停/继续游戏
- ✅ 游戏结束后可重新开始
