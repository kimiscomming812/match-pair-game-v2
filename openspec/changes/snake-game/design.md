# Design: Snake Game (贪食蛇游戏)

## 技术栈
- **HTML5/CSS3** - 游戏界面与样式
- **原生 JavaScript** - 游戏逻辑控制

## 核心游戏逻辑
1. **游戏状态管理**:
   - `snake`: 蛇身坐标数组（每个元素为 {x, y}）
   - `food`: 食物坐标 {x, y}
   - `direction`: 当前移动方向
   - `score`: 当前分数
   - `gameRunning`: 游戏运行状态
   - `gameSpeed`: 游戏速度

2. **游戏循环**:
   - 使用 `setInterval` 控制游戏主循环
   - 每次循环：移动蛇 → 检查碰撞 → 检查吃食物 → 更新显示

3. **控制逻辑**:
   - 键盘事件监听（方向键）
   - 防止反向移动（如向右时不能立即向左）

## 游戏界面
- **游戏区域**: 20x20 网格，每个格子 20px
- **信息栏**: 显示分数、最高分、游戏状态
- **控制按钮**: 开始/暂停、重新开始
